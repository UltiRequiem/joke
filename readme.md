# The Joke API

![CI](https://github.com/UltiRequiem/joke/workflows/CI/badge.svg)

## API

### `GET /`

> https://joke.deno.dev

Returns a random joke in this format:

```json
{
  "id": 229,
  "type": "general",
  "setup": "What do you call an alligator in a vest?",
  "punchline": "An in-vest-igator!"
}
```
